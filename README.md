# KCD2 Dice — браузерная игра в кости на двоих по LAN

Веб-реализация игры в кости из Kingdom Come: Deliverance II. Два игрока, локальная сеть (или Radmin VPN), браузер.

## Правила

6 кубиков. Бросаешь — откладываешь зачётные комбинации — решаешь: рискнуть и перебросить или зафиксировать очки.

**Комбинации:**
- `1` = 100, `5` = 50
- 3 одинаковых: 3×1 = 1000; 3×N (N=2..6) = N × 100
- 4/5/6 одинаковых удваивают тройку: ×2 / ×4 / ×8
- Стрит 1-5 = 500, 2-6 = 750, 1-6 = 1500
- **Зонк:** ноль очков в броске → все очки хода сгорают
- **Hot dice:** все 6 костей зачётные → обязательный переролл всех 6
- Победа: первый, у кого после фиксации `>= target` (2000 / 3000 / 4000 на выбор)

## Запуск

Требуется Node.js 20+.

```bash
npm install
npm start                  # сервер на http://localhost:8080
npm start -- --port 9000   # другой порт
npm test                   # юнит-тесты движка
```

## Игра с товарищем по Radmin VPN

1. Установите [Radmin VPN](https://www.radmin-vpn.com/) на обеих машинах, подключитесь к одной сети.
2. Хост запускает `npm start`, узнаёт свой Radmin-IP (в окне Radmin рядом с именем сети).
3. Хост открывает `http://localhost:8080` в браузере.
4. Второй игрок открывает `http://<radmin-ip-хоста>:8080`.
5. Оба вводят имена, хост выбирает цель (2000/3000/4000), нажимает Start.

## Технологии

- Node.js + Express (статика) + `ws` (WebSocket)
- Vanilla HTML/CSS/JS (без фреймворков, без бандлера)
- `node:test` для юнит-тестов
